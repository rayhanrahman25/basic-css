What is Flexbox?

Flexbox is a CSS layout module that makes it easier to design flexible and responsive layouts. It works with a container (flex container) and items (flex items).

Note: Flexbox is ideal for arranging elements in one dimension (either as a row or a column).

Flex Container Properties

These properties are applied to the parent container to control the layout of its children.

1. Flex Direction

Defines the direction of flex items:

flex-direction: row | row-reverse | column | column-reverse;

Flex Item Properties

These properties are applied to the child elements to control their behavior within the flex container.

1. Flex Grow

Defines how much a flex item can grow relative to other items:

flex-grow: number;
